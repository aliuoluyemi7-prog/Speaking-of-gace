# Speaking of Grace

Grace for the thinking Christian — a static site (HTML/CSS/JS, no build step) covering:

- `index.html` — Home
- `about.html` — About
- `articles.html` — Articles / blog
- `books.html` — Books & resources library
- `projects.html` — Chinexo, CashinSight, ExamWiseNG, and more
- `resources.html` — Resources for teachers, students, churches, entrepreneurs
- `contact.html` — Contact form

## Deploying

No build step required — this is a static site.

**Netlify:** New site from Git → pick this repo → leave build command blank → publish directory `/`

**Vercel:** New Project → pick this repo → Framework Preset: "Other" → leave build command blank → output directory `/`
